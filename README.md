# Rao Cattle Farm Frontend

A modern React and Vite frontend for the Rao Cattle Farm website. The application includes a polished landing page experience with sections for hero content, about, products, and contact details, along with a basic admin area.

## Features

- Responsive single-page style homepage
- Light and dark theme support
- Product showcase section
- Contact section for customer outreach
- Admin login and portal route
- Fast development workflow with Vite

## Tech Stack

- React 18
- Vite
- React Router DOM
- React Swipeable

## Getting Started

### Prerequisites

Make sure you have installed:

- Node.js 18 or later
- npm

### Installation

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Routes

- `/` — Main website
- `/admin` — Admin login and portal
